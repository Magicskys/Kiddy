 被动式SQL扫描工具
   
 不得用于非法用途
 

 **使用说明**

      使用前进入Sqlmap输入以下命令
            python sqlmapapi.py -s
      
      浏览器代理端口号为8080
      
      在程序内输入start开启嗅探,stop停止嗅探
      
      程序内输入help显示帮助菜单
